# RoboSpeaker

RoboSpeaker is a simple Python command-line application for **macOS** that converts text into speech using Apple's built-in `say` command. Designed for simplicity, it allows users to type any message and hear it spoken aloud instantly.

---

# Prerequisites

Before running the project, make sure you have:

* Python 3.x installed
* macOS (the project uses the built-in `say` command)

You can verify your Python installation by running:

```bash
python3 --version
```

---

# Installation

### Clone the repository

```bash
git clone https://github.com/manavexists/RoboSpeaker.git
```

### Navigate to the project directory

```bash
cd RoboSpeaker
```

---

# Usage

Run the application:

```bash
python3 main.py
```

After launching, enter any text you want your Mac to speak.

Example:

```text
Welcome to RoboSpeaker

What do you want me to speak?
Hello, everyone!
```

To exit the application, type:

```text
q
```

---

# Features

* Convert text into speech
* Continuous command-line interaction
* Exit anytime by typing `q`
* Lightweight and beginner-friendly
